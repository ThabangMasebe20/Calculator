# Calculator
A basic calculator desktop app built with Python and Tkinter
Supports basic arithmetic operations with a clean graphical interface.

## Features
- Addition, subtraction, multiplication, and division
- Clear and delete functionality
- Keyboard and button input support
- Error handling for invalid expressions (e.g. division by zero)

## Tech Stack
- Python
- Tkinter

## How to Run
1. Make sure Python is installed
2. Run: `python calculator.py`
